# Nextpilot-VScode-Extension

## 开发指南

请参照：<https://rackar.github.io/vscode-ext-doccn/>

### 安装 nodejs

安装最新长期支持版本 [nodejs](https://nodejs.cn/download/)，比如

[node-v20.18.0-x64.msi](https://npmmirror.com/mirrors/node/v20.18.0/node-v20.18.0-x64.msi)

切换到国内NPM镜像源

```shell
npm config set registry https://npm.aliyun.com
```

安装 Yeoman 和 VScode Extension Generator：

```shell
npm install -g yo generator-code
```
